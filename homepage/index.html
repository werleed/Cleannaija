A cleaner Nigeria, one pickup at a time

Request pickups, report sanitation issues to local authorities, verify your account with OTP, and earn from recyclable waste — all through a simple Telegram bot.

Start on Telegram

How it works

  1. Verify your phone with Twilio OTP (test mode available).
  2. Scan or describe waste — the bot simulates detection and weight.
  3. Admin confirms collection and credits the account automatically.
  4. Request withdrawal; admin approves and simulates payout.

Core features

  • ✅ Phone verification (Twilio)
  • ✅ Waste scan and weight detection (simulated)
  • ✅ Referral & rank system
  • ✅ Admin-managed approvals & simulated payouts
  • ✅ Lightweight storage (local JSON files)

Rate & Earnings

Default payout: ₦120 per kg — editable by admin via /setrate.
